HTML Cheatsheet

HTML Elements


Tag Name Purpose
< a > Anchor Used to create hyperlinks in an HTML document.
< body > Body Represents the content that will displayed on the web browser.
< div > Divison Used as a container to divide an HTML document into sections.
< em > Emphasis Emphasizes an element, typically renedered as italics.
< br > Line Break Creates a line break in the text, and has no closing tag.
< li > List Item Creates a list item in an ordered or unordered list.
< head > Head Metadata containing general information about the HTML page.
< h1-h6 > Headings The six different levels of headings from largest h1 to smallest h6.
< html > HTML The root of an HTML document, all html content between its openeing and closing tags.
< img > Image Used to embed images using a src attribute.
< ol > Ordered List Creates a list in nyumbered sequential order.
< p > Paragraphs Contains and displays a block of text.
< span > Span An inline container used to group text for styling purposes.
< strong > Strong Highlights important text, typically rendered as bold.
< title > Title The title is displayed in the browser's window, only written within the < head >.
< ul > Unordered List Creates a list item in no particular order using bullet points as its default.
< video > Video Embeds a video player using src and contorls attributes.
